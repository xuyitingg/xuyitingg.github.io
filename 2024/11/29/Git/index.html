Git相关操作

Git官网 https://git-scm.com/downloads

安装过程到Adjusting your PATH environment步骤时选择Use Git from Git Bash only

设置用户名:
git config – global user.name ‘github上注册的用户名’
git config – global user.email ‘注册时候的邮箱’

用户身份切换:
不用的远程仓库不同账号可能使用的git账号不同
在同一台电脑上进行git操作时需要先进行如下步骤
控制面板-用户账户-管理windows凭据
找到git信息并删除 这样在进行git操作时会提示输入用户账号和密码进行用户切换

连接远程分支:
如果是通过clone的可以直接输入以下命令查看已连接远程仓库信息
git remote -v
如果不是clone的需要先初始化
git init
git remote add origin https
删除本地分支
git branch -D xxx
查看远程仓库所有分支
git ls-remote –heads origin
新建本地分支并切换
git checkout -b xuyiting
新建远程分支并连接
git add .
git commit -m “new branch”
git push -u origin xuyiting

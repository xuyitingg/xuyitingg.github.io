Windows相关操作

右键菜单集成打开方式

  1. WIN+R 输入 regedit 打开注册表编辑器
  2. HKEY_CLASSES_ROOT/*/shell
    右键shell新建项命名为OpenWithWhat 默认数据为OpenWithWhat
    右键OpenWithWhat新建字符串 值为Icon 数值数据为.exe文件路径
    右键OpenWithWhat新建项command 默认数据为”.exe文件路径” “%1”
  3. HKEY_CLASSES_ROOT/Directory/shell
    右键shell新建项命名为OpenWithWhat 默认数据为OpenWithWhat
    右键OpenWithWhat新建字符串 值为Icon 数值数据为.exe文件路径
    右键OpenWithWhat新建项command 默认数据为”.exe文件路径” “%V”
  4. HKEY_CLASSES_ROOT/Directory/Background/shell
    右键shell新建项命名为OpenWithWhat 默认数据为OpenWithWhat
    右键OpenWithWhat新建字符串 值为Icon 数值数据为.exe文件路径
    右键OpenWithWhat新建项command 默认数据为”.exe文件路径” “%V”

WIN11改WIN10右键菜单

  1. WIN+X 选择Windows终端(管理员)
  2. reg add “HKCU\Software\Classes\CLSID{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32” /f /ve
  3. 打开任务管理器 右键Windows资源管理器 重新启动

系统盘还原

  1. WIN+R 输入 diskpart 打开终端
  2. 输入 list disk 显示磁盘列表
  3. sel dis 3 选中 U 盘
  4. clean
  5. cre par pri 新建分区
  6. format fs=ntfs quick 格式化
  7. assign letter x
